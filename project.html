Lahore Car Rentals

This project features a user-friendly prototype for a fictional car rental service in Lahore, designed to streamline local vehicle booking with a focus on ease of use and modern design.

Created during ITEC 3230 (Designing User Interfaces) as part of a team assignment emphasizing usability, feedback integration, and interface prototyping.
